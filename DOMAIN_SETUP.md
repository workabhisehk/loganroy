# Domain setup for loganroy.in

This portfolio should point to `loganroy.in`.

## If deploying on Netlify

- Set the production domain to `loganroy.in`.
- Add `www.loganroy.in` as a domain alias if needed.
- Ensure the DNS has a `CNAME` or `A` record that points to Netlify.

## If deploying elsewhere

- Point `loganroy.in` to the hosting provider with the required DNS records.
- Add a redirect for `www.loganroy.in` if you want a single canonical domain.
